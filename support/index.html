Need help with NoFlo?

NoFlo is a Flow-Based Programming environment built together by Flowhub team and a number of open source contributors.

Tweets by @noflo

Stack Overflow

The Stack Overflow website is great for posting and answering technical questions. You can browse through the existing questions or ask your own. Make sure to utilize the noflo tag!

Social Media

If you want to follow NoFlo developments, we're available in several social media services:

  • Twitter
  • Facebook
  • Reddit /r/DataflowProgramming

Mailing List

There is a general Flow-Based Programming mailing list that occasionally features NoFlo discussion.

Consulting Services

Flowhub UG provides consulting services for NoFlo and other flow-based programming systems.

NoFlo - Flow-Based Programming for JavaScript is free software developed by Henri Bergius and backed via Kickstarter.
